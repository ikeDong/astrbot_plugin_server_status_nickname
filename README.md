# AstrBot 群昵称动态显示服务器状态插件

灵感来源：[长安某](https://github.com/zgojin/astrbot_plugin_botName)

通过群昵称动态显示服务器 CPU 和内存状态，并提供历史折线图生成。群组收到消息后自动注册，定时器持续更新昵称，无需等待对话触发。


## 修改说明

该插件基于 https://github.com/DITF16/astrbot_plugin_server_status_nickname 修改。

本分支新增功能：

- 新增休息时段消息拦截：当休息模式开启且当前处于休息时间时，普通消息会被拦截，不再进入 LLM。
- 新增指令放行开关：休息时段默认仍放行以 `/` 或 `／` 开头的指令，避免无法调整插件配置。
- 新增静默或自定义提示：可通过 `rest_block_reply` 设置拦截时回复，留空则静默拦截。
- 保留原有休息时段固定昵称功能，可在休息时显示固定昵称文案。

### 新增配置项

| 配置 | 说明 | 默认值 |
|---|---|---|
| `rest_block_messages` | 休息时段是否拦截普通消息进入 LLM | `true` |
| `rest_allow_commands` | 休息时段是否放行 `/` 或 `／` 开头的指令 | `true` |
| `rest_block_reply` | 消息被拦截时发送的提示，留空则静默拦截 | 空 |

## 安装

在 AstrBot 管理面板的插件市场输入本仓库地址安装，或上传 zip 包安装。插件依赖会在加载时自动安装。

如果自动安装失败，请参考[依赖手动安装](#依赖手动安装)。

### 中文字体配置

图表中的中文需要字体文件才能正常显示，否则会显示为方块。将任意中文字体文件重命名为 `font.ttf`，放入插件目录下的 `font/` 文件夹中即可。

## WebUI 配置

在 AstrBot 管理面板 → 插件配置中可配置以下项目：

### 群组过滤模式

选择 `白名单模式` 或 `黑名单模式`：

- **白名单模式**：仅对列表中的群聊生效。列表为空时所有群聊都生效
- **黑名单模式**：除列表中的群聊外，其他所有群聊都生效

### 群组列表

填入需要过滤的群号。在 WebUI 中以列表形式配置，一行一个纯数字群号，例如：

```
123456789
987654321
```

**白名单模式示例：**
> 列表填入 `123456789`、`987654321`
> → 仅这两个群会显示状态，其他群不受影响

**黑名单模式示例：**
> 列表填入 `123456789`
> → 除该群外，其他所有群都会显示状态

**如何获取群号：**
- QQ 群：打开群设置 → 群号
- 对机器人发送 `/sid` 指令，返回的「会话 ID」末尾数字即群号（如 `aiocqhttp:GroupMessage:123456789` 中的 `123456789`）

### 昵称状态文案模板

自定义昵称后缀中显示的内容，支持以下变量（可任意排列组合）：

| 变量 | 含义 | 示例值 |
|---|---|---|
| `{cpu_percent}` | CPU 占用百分比（纯数字） | `5.1` |
| `{mem_percent}` | 内存占用百分比（纯数字） | `45.2` |
| `{mem_used_gb}` | 已用内存（纯数字 GB） | `7.4` |
| `{mem_total_gb}` | 总内存（纯数字 GB） | `15.9` |
| `{mem_used_mb}` | 已用内存（纯数字 MB） | `7577` |
| `{mem_total_mb}` | 总内存（纯数字 MB） | `16384` |

> 以上变量均为纯数字，如需显示单位请在模板中自行添加，如 `{cpu_percent}%`、`{mem_used_gb}GB`。

插件会在模板文案外自动包裹 `()`。

**默认模板：**
```
理智使用{cpu_percent}% 脑容量使用{mem_percent}%
```
效果：`(理智使用5.1% 脑容量使用45.2%)`

**自定义示例：**
```
CPU:{cpu_percent}% 内存:{mem_used_gb}/{mem_total_gb}GB
```
效果：`(CPU:5.1% 内存:7.4/15.9GB)`


### 休息时段固定昵称

可在 WebUI 中开启 `休息时段固定昵称显示`。开启后，指定时间段内不再读取并展示 CPU/内存状态，而是显示固定文案。

配置项：

| 配置 | 说明 | 示例 |
|---|---|---|
| `rest_mode_enabled` | 是否启用休息时段固定显示 | `true` |
| `rest_start_time` | 休息开始时间，格式 HH:MM | `23:00` |
| `rest_end_time` | 休息结束时间，格式 HH:MM | `07:00` |
| `rest_fixed_text` | 休息时段显示的固定文字 | `Zz休息中~` |

支持跨天时间段，例如 `23:00` 到 `07:00`。
效果：`| Zz休息中~`。

### 定时更新间隔

定时器自动更新昵称的间隔时间（1-60 分钟），默认 5 分钟。

## 指令

| 指令 | 说明 |
|---|---|
| `/昵称显示开启 群号` | 将群加入列表并立即更新昵称 |
| `/昵称显示关闭 群号` | 将群从列表移除并清理昵称 |
| `/服务器更新` | 立即记录状态并更新所有群昵称 |
| `/改变脑容量显示` | 在百分比和绝对值显示间切换 |
| `/理智记录 n小时/天` | 查看最近 n 小时/天的 CPU 折线图 |
| `/脑容量记录 n小时/天` | 查看最近 n 小时/天的内存折线图 |
| `/刷新缓存` | 刷新当前群聊的昵称状态 |

> 指令 `/昵称显示开启`、`/昵称显示关闭` 会同步修改 WebUI 配置中的群组列表。

## 注意事项

- 状态数据保留 10 天，每天凌晨 4:00 自动清理
- 首次使用需在群中发送任意消息以注册群组，之后定时器会自动持续更新
- 插件通过在群昵称末尾追加 `(状态文案)` 显示状态，更新时自动替换旧后缀
- **注意**：插件会识别并替换昵称末尾所有 `(...)` 格式的后缀，请勿在昵称末尾使用括号以免被覆盖

### <a id="依赖手动安装"></a>依赖手动安装

如果自动安装失败（插件日志中出现 `ModuleNotFoundError: No module named 'matplotlib'`），需要手动安装依赖 `matplotlib`，根据部署方式选择：

**方式一：WebUI（推荐）**

1. 打开 AstrBot 管理面板
2. 进入 **平台日志** 页面
3. 点击右上角 **「安装 pip 库」** 按钮
4. 输入 `matplotlib`，点击安装
5. 等待提示安装成功后重启 AstrBot

**方式二：Docker 环境**

```bash
docker exec -it <容器名> pip install matplotlib
docker restart <容器名>
```

**方式三：源码部署（uv）**

```bash
cd AstrBot
uv pip install matplotlib
```

**方式四：源码部署（venv）**

```bash
cd AstrBot
# Linux / macOS:
source venv/bin/activate
# Windows:
# venv\Scripts\activate
pip install matplotlib
```
